# 📚 BookToText — Конвертер электронных книг в чистый текст

[![Python Version](https://img.shields.io/badge/python-3.10%20%7C%203.11%20%7C%203.12%20%7C%203.13-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-macOS-lightgrey.svg)](#)

**BookToText** — это удобное настольное (GUI) приложение для macOS, написанное на Python с использованием библиотеки **CustomTkinter**. Оно предназначено для пакетной или одиночной конвертации электронных книг и документов различных форматов в чистый, структурированный текст формата `.txt` в кодировке UTF-8.

Приложение спроектировано с акцентом на стабильность работы, многопоточность (GUI никогда не блокируется во время конвертации) и высокое качество очистки извлекаемого текста (без HTML-тегов, скриптов, лишних пустых строк и битых символов).

## ✨ Ключевые возможности

*   🎨 **Современный интерфейс:** Построен на базе `CustomTkinter` с полноценной темной темой, отзывчивыми элементами управления и динамическими анимациями/индикаторами.
*   🧵 **Многопоточная обработка:** Конвертация файлов запускается в фоновом потоке. Вы можете управлять очередью, просматривать логи и перемещать окно прямо во время работы.
*   🛡️ **Изоляция ошибок:** Если один из файлов в очереди поврежден или защищен DRM, приложение залогирует ошибку и перейдет к следующему файлу без сбоя всей очереди.
*   ⚡ **Умная очистка текста:**
    *   Нормализация переносов строк (`\r\n` $\to$ `\n`).
    *   Удаление дублирующихся пустых строк (не более двух подряд для читаемости).
    *   Очистка от лишних пробелов на концах строк.
    *   Удаление HTML-разметки, CSS-стилей, JavaScript-вставок, метатегов и встроенных скриптов.
*   💾 **Предотвращение перезаписи:** При совпадении имен выходных файлов автоматически добавляется числовой индекс (`книга_1.txt`, `книга_2.txt`).
*   🔍 **Эвристическая валидация полноты:** Интеллектуальный алгоритм проверяет результат перед сохранением. Если размер выходного текста слишком мал относительно исходного файла (критически низкое соотношение) или текст обрывается, выдается предупреждение/ошибка, гарантируя, что книга конвертирована полностью.
*   **Офлайн-работа:** Приложение работает исключительно локально, ваши файлы никуда не отправляются.
*   **Два формата вывода:** Выбирайте между чистым текстом (`.txt`) или размеченным форматом Markdown (`.md`), который сохраняет заголовки, списки, курсив и жирность шрифта.
*   **Умное переименование:** Автоматическое извлечение автора и названия книги из метаданных (или разбор оригинального имени) для красивого сохранения файлов в формате `Автор - Название книги.ext`.
*   **Пакетная конвертация:** Выбирайте отдельные файлы, группу файлов или целую папку (поддерживается рекурсивный поиск файлов внутри вложенных папок).

## 📖 Поддерживаемые форматы и парсеры

| Формат | Библиотека парсинга | Особенности реализации |
| :--- | :--- | :--- |
| **EPUB** | `EbookLib` + `BeautifulSoup4` | Порядок извлечения соответствует логической структуре книги (через `spine`), используется устойчивый к битой разметке парсер `html.parser`. |
| **FB2 / FB2.ZIP** | `xml.etree` (стандартная) | Быстрый XML-анализ структуры, автоматическая распаковка `.zip` архивов без сторонних утилит. |
| **PDF** | `pypdf` | Постраничное чанковое чтение для экономии оперативной памяти при обработке огромных документов. |
| **DOCX** | `python-docx` | Извлечение текста с сохранением структуры абзацев. |
| **MOBI / AZW3** | `mobi` + `BeautifulSoup4` | Чтение незашифрованных MOBI-файлов, автоматическая проверка DRM. |
| **RTF** | `striprtf` | Полная очистка от служебных тегов RTF-разметки. |
| **HTML / HTM** | `BeautifulSoup4` | Извлечение чистого текстового содержимого с удалением управляющих тегов, скриптов, стилей и фреймов. |

## 🛠️ Установка и запуск

### ⚠️ Системные требования для macOS
Для корректной отрисовки интерфейса Tkinter на macOS рекомендуется установить поддержку Tcl/Tk. Сделайте это через Homebrew:
```bash
brew install python-tk@3.13
```

### 1. Клонирование и виртуальное окружение
```bash
# Клонируйте репозиторий
git clone https://github.com/aplaygs/BookToText.git
cd BookToText

# Создайте и активируйте виртуальное окружение
python3 -m venv venv
source venv/bin/activate
```

### 2. Установка зависимостей
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### 3. Запуск приложения
```bash
python main.py
```

## 🧪 Тестирование

Проект покрыт комплексными юнит-тестами, проверяющими логику всех конвертеров, утилит очистки, обработку некорректных файлов, проверку полноты и обход директорий.

Для запуска тестов используйте:
```bash
python test_runner.py
```
*Все 25 тестов должны завершиться успешно (OK).*

## 📦 Сборка автономного приложения (.app для macOS)

Вы можете собрать приложение в виде единого исполняемого пакета `.app` для macOS с помощью **PyInstaller**:

```bash
# Сборка пакета с иконкой
pyinstaller --windowed --name "BookToText" --icon=icon.icns main.py
```

После завершения сборки готовое приложение появится в папке `dist/BookToText.app`. Вы сможете запускать его двойным кликом без необходимости открывать терминал.

### 🚀 Автоматическое обновление и деплой
Для того чтобы автоматически прогнать тесты, собрать `.app` и переустановить приложение в папку `/Applications/` вашей системы, воспользуйтесь скриптом:
```bash
source venv/bin/activate
./deploy.py
```
Скрипт проверяет безопасность, корректность работы (юнит-тесты) и самостоятельно устанавливает свежую версию приложения в систему.
